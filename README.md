# CCRM Java Project

## 📌 Description
This project is a Java-based Customer/College Record Management System (CCRM). It is designed to manage records efficiently using core Java concepts like OOP, file handling, and collections.

## 🚀 Features
- Add, update, and delete records
- View stored data
- Menu-driven interface
- Efficient data handling

## 🛠️ Technologies Used
- Java
- OOP Concepts
- File Handling / Collections

## 📂 Project Structure
- Source code files
- Supporting classes
- Data handling modules

## ▶️ How to Run
1. Compile the program:
   javac Main.java

2. Run the program:
   java Main
