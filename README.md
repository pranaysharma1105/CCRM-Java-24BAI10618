# CCRM Java Project

##  About the Project

This project is a simple yet practical Java-based Record Management System developed to understand how real-world data can be organized and managed efficiently.

It focuses on applying core Java concepts like object-oriented programming, collections, and basic data handling in a structured way. The system provides a menu-driven interface where users can interact with records easily.

---

##  What This Project Does

The application allows users to manage records through a console interface. It is designed to simulate how basic management systems work in real-life scenarios.

You can:

* Add new records
* View existing records
* Update details when needed
* Delete records
* Navigate through a simple menu system

---

##  Concepts Used

While building this project, the following Java concepts were applied:

* Object-Oriented Programming (Classes and Objects)
* Encapsulation
* Collections (like ArrayList)
* File Handling (if used in your code)
* Exception Handling

## 🛠️ Tech Stack

* Java
* Console-based Interface

##  How to Run the Project

1. Open terminal or command prompt

2. Navigate to the project folder

3. Compile the program:
   javac Main.java

4. Run the program:
   java Main

---

##  Project Structure

The project is organized into multiple Java files, each responsible for handling different parts of the system such as data storage, logic, and user interaction.
